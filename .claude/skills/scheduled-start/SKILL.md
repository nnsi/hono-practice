# 時刻指定で作業開始

ユーザーが「HH:MMに〜して」「HH:MM になったら〜」と依頼した場合、指定時刻まで待機してから作業を実行する。

## 手順

### Step 1: 時刻とタスクを確認

ユーザーの発言から以下を抽出する:
- **開始時刻**: HH:MM形式
- **作業内容**: 待機後に実行するタスク

### Step 2: 待機実行

待機時間に応じてBashツールの呼び出し方を変える:

#### 10分以内の場合

```bash
node scripts/wait-until.js HH:MM
```

通常実行（`run_in_background` なし）で待つ。完了後すぐに作業開始。

#### 10分を超える場合

```bash
node scripts/wait-until.js HH:MM
```

`run_in_background: true` で実行する。Bashツールのtimeoutが最大10分のため、通常実行だとタイムアウトで強制終了される。

バックグラウンド実行の完了通知が届いたら作業を開始する。

### Step 3: 作業開始

待機完了後、Step 1で確認した作業内容を実行する。通常の開発フローに従う。

## 注意事項

- 今日の時刻を過ぎている場合は翌日として扱われる（スクリプト側で処理済み）
- 5分ごとに残り時間がログに出る
- ユーザーがCtrl+Cで待機をキャンセルできる
