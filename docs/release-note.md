# Actiko v2 リリースノート

v2リリース日: 2026-02-25

`docs/diary/` をもとに、日記の日付をリリース日として扱い、ユーザー向けの変更を中心に整理しています。内部リファクタリングや開発運用メモは省略しています。

## ハイライト

- オフラインファースト体験と同期の信頼性を大幅に改善
- 記録モードを拡張し、活動に合わせた入力方法を選べるように改善
- 目標管理を強化し、曜日別目標やDebt関連機能を追加
- タスクとアクティビティの連携を強化
- モバイルの認証、入力、画像選択、日付選択、OTA体験を改善
- ホーム画面ウィジェット、Pro基盤、音声ショートカット、英語対応、ダークモードを追加
