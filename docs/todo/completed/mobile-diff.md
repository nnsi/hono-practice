# Web / Mobile Function Diff

現時点で確認できる、Web と Mobile の機能差一覧。

## 1. 起動時の初期画面切り替え

- Web は `showGoalOnStartup` 設定に応じて起動時に Goals 画面へ遷移できる
- Mobile は設定項目は存在するが、起動時導線には反映されていない

## 2. Google アカウント連携

- Web は設定画面から Google 連携状態の確認ができる
- Web は設定画面から Google アカウント連携を実行できる
- Mobile は Google ログインはあるが、設定画面での連携状態表示と追加連携 UI がない

## 3. CSV インポート機能の深さ

- Web は CSV カラムのマッピングができる
- Web は import 前 preview ができる
- Web は preview 内で行編集・削除ができる
- Web は CSV テンプレートのダウンロードができる
- Mobile は簡易 import のみで、列マッピング・行編集・テンプレート配布がない

## 4. アクティビティ作成時のアイコン設定

- Web はアクティビティ作成時に emoji / upload の選択ができる
- Mobile はアクティビティ作成時は emoji のみ
- Mobile は編集時のみ画像アップロード対応で、作成時の機能が揃っていない
