# リフレッシュトークン実装タスクリスト

## 概要

- アクセストークン有効期限: 15分
- リフレッシュトークン有効期限: 1ヶ月
- アクセストークン: JWT形式
- リフレッシュトークン: DB保存
- トークン無効化: ログアウト時のみ
